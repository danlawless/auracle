## Emergency Fixes

### If Amazon Shows Formatting Issues
1. Download the "Look Inside" preview
2. Identify problems
3. Fix in source file
4. Re-upload manuscript
5. Wait for re-review (24-48 hours)

### If Gumroad Links Break
1. Check URL hasn't changed
2. Update all Linktree/bio links
3. Test on mobile

### If No Reviews After 2 Weeks
1. Personal outreach to 10 more people
2. Offer free copy in exchange for honest review
3. Post "looking for reviewers" on social
4. Check that review link is in back of book

### If Content Isn't Getting Engagement
1. Review what competitors post
2. Try different formats (carousel vs. single image)
3. Post at different times
4. Ask audience what they want

### If a Volume Launch Underperforms
1. Extend launch week to 2 weeks
2. Run a limited-time discount
3. Bundle with previous volumes
4. Increase personal outreach
5. Don't panic—momentum builds over time
